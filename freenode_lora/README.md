# FreeNode v0.2 — Dual Transport: ESP-NOW + LoRa

## Что нового в v0.2

**Мультитранспорт** — первая реализация ключевого принципа FreeNode.
Две платы теперь общаются по двум каналам одновременно:

| Транспорт | Скорость | Дальность (город) | Роль |
|-----------|----------|--------------------|------|
| ESP-NOW (Wi-Fi 2.4GHz) | ~1 Mbps | 50–150 м | Основной, быстрый |
| LoRa SX1262 (869MHz) | ~3.5 kbps (SF9) | 250–1000 м | Резервный, дальний |

**Автовыбор транспорта:**
- Если peer отвечал по ESP-NOW в последние 10 сек → ESP-NOW
- Иначе → LoRa (fallback)
- Переключение прозрачно

**Идентификация:**
- Каждый пакет содержит последние 2 октета MAC отправителя
- На OLED: `[W]A3:F1> Hello` (W=ESP-NOW) или `[L]A3:F1> Hello` (L=LoRa)

## Платформа
- **Плата:** LILYGO T3S3 V1.2 (ESP32-S3 + SX1262 + SSD1306 OLED)
- **LoRa:** SX1262, 869 MHz, SF9, +22 dBm
- **ESP-NOW:** 2.4 GHz, канал 1, broadcast

## Установка

Те же библиотеки что в v0.1:
1. **RadioLib** by Jan Gromes
2. **Adafruit SSD1306**
3. **Adafruit GFX Library**

WiFi и ESP-NOW встроены в ESP32 — ничего дополнительно ставить не надо.

## Настройки Arduino IDE

- Board: **ESP32S3 Dev Module**
- USB CDC On Boot: **Enabled** ← ОБЯЗАТЕЛЬНО
- Upload Speed: 115200
- Flash Size: 4MB

## Прошивка

**ВАЖНО: прошить ОБЕ платы этим скетчем!**

1. Подключи USB-C
2. Если порт не появляется → зажми BOOT, включи плату, отпусти BOOT
3. Upload

## OLED экран

```
FreeNode v0.2 [A3:F1]     ← мой ID (последние 2 октета MAC)
TX:ESP-NOW  E:5 L:2       ← текущий транспорт, счётчики
─────────────────────
[W]B7:22> Привет          ← [W]=ESP-NOW, [L]=LoRa, ID отправителя
[L]B7:22> Далеко ушёл     ← автоматически переключился на LoRa
>PING via LoRa            ← мой PING
[W]B7:22> PONG            ← ответ
```

## Команды Serial Monitor

| Команда | Действие |
|---------|----------|
| `текст` | Отправить сообщение |
| `/ping` | Отправить PING (получатель автоответит PONG) |
| `/status` | Показать статистику транспортов |
| `/lora` | Принудительно LoRa (для тестов дальности) |
| `/auto` | Вернуть автовыбор транспорта |

## Тест дальности с другом

### Этап 1: ближняя зона (0–100 м)
1. Прошей обе платы
2. Друг уходит — на экранах видно `[W]` (ESP-NOW)
3. Пиши текст, жми BOOT для PING

### Этап 2: переключение (~100–200 м)
4. Друг уходит дальше — ESP-NOW пропадает
5. Через 10 сек автоматический fallback на LoRa
6. На экране `[W]` сменится на `[L]` — это оно!

### Этап 3: дальняя зона (200+ м)
7. Только LoRa
8. `/lora` — принудительно LoRa для чистого теста
9. Записывай RSSI и SNR — покажут запас

## Формат пакета FreeNode v0.2

```
Байт  Назначение
[0]   0xFE — magic byte
[1]   0x02 — версия протокола
[2]   тип: 0x01=TEXT, 0x02=PING, 0x03=PONG
[3-4] последние 2 октета MAC отправителя
[5]   длина payload
[6..] payload (текст, UTF-8)
```

Одинаковый формат для LoRa и ESP-NOW — транспорт прозрачен для протокола.

## Параметры LoRa

| Параметр | Значение | Описание |
|----------|----------|----------|
| Частота | 869 MHz | RU868 разрешённый |
| SF | 9 | Баланс дальности/скорости |
| BW | 125 kHz | Стандартная полоса |
| TX Power | 22 dBm | Максимум SX1262 |
| Sync Word | 0x12 | FreeNode |
